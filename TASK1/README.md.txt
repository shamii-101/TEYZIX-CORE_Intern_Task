Student Performance Analytics System

Short Description:
This project is a Python-based Student Performance Management System. It manages student records using CSV files and provides functionalities like adding, updating, deleting, viewing students, and calculating performance analytics such as total marks, average, percentage, and grades.

Features:
- Add new student records
- Update existing student marks
- Delete student records
- View student data
- Calculate performance (Total, Average, Percentage, Grade)

Technologies Used:
- Python
- Pandas
- CSV File Handling

How to Run:
1. Make sure Python is installed on your system.
2. Install pandas library if not installed:
   pip install pandas
3. Place all files (main.py and data.csv) in one folder.
4. Open terminal in project folder.
5. Run the project using command:
   python main.py
