# Smart Invoice Processing System

## Project Overview

Smart Invoice Processing System is a Python-based application that manages invoice records using SQLite database. The system allows users to add, search, update, delete and analyze invoice data.

The project also provides invoice export, logging, PDF text extraction and reporting features.

---

## Features

### Invoice Management (CRUD)

- Add new invoices
- Display all invoices
- Search invoices
- Update invoice details
- Delete invoices


### Database Management

- SQLite database integration
- Automatic table creation
- Data storage and retrieval


### Validation

- Duplicate invoice checking
- Required field validation
- Amount validation


### Logging System

- Records system activities
- Maintains error logs
- Generates invoice_system.log file


### Export System

- Export invoices into CSV format
- Export invoices into Excel format


### PDF Processing

- Extract text from PDF invoices


### Reporting

- Total invoice count
- Total invoice amount
- Payment status report
- Vendor based report
